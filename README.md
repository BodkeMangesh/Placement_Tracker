# Placement Tracker

A Placement Tracker built using FastAPI, SQLite, and Playwright.

## Features

* Save Jobs
* View All Jobs
* Search by Company
* Search by Location
* Combined Filters
* Duplicate Prevention
* SQLite Database Integration

## Tech Stack

### Backend

* Python
* FastAPI
* SQLite
* Playwright

### Frontend

* React / Next.js (Upcoming)

## API Endpoints

### Jobs

GET /jobs

GET /jobs/search?company=TCS

GET /jobs/location?location=Pune

GET /jobs/filter?company=TCS&location=Pune

### Scraper

GET /save-jobs
